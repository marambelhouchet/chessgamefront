⚔ Conquête des Territoires ⚔

Règles du jeu
Comment jouer
  • 🎲 Lancez le dé pour obtenir vos points de mouvement (1–6).
  • 👆 Cliquez une unité pour la sélectionner, puis une case cible.
  • ➡ Mouvement en ligne droite (horizontale ou verticale) uniquement.
  • ⚔ Attaque : chaque joueur lance un dé + force. Le plus haut gagne.
  • ✦ Case Bonus : +1 force au combat pour l'occupant.
  • ⚠ Piège : l'unité qui entre est détruite immédiatement.
  • 🏆 Victoire : contrôler 33 cases ou éliminer toutes les unités adverses.

Combat détaillé
  • 1. Attaquant : lance dé + force de son unité (+ bonus si sur case Bonus).
  • 2. Défenseur : même calcul.
  • 3. Le score le plus élevé l'emporte. En cas d'égalité, le défenseur gagne.
  • 4. Victoire attaquant → capture la case. Défaite → son unité est détruite.
